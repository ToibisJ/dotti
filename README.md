# Dotti App Demo

אפליקציית דמו אינטראקטיבית בעברית עבור Dotti: עוזרת להורים לנהל משימות יומיומיות, תכנון יום, צ׳ק אין רגשי וצ׳אט תמיכה.

## תצוגה מקומית

אפשר לפתוח את `index.html` בדפדפן, או להריץ שרת סטטי:

```powershell
npx http-server . -p 4173
```
